## Frequently Asked Questions

### How do I install this skill?

Copy the skill directory to your Claude Code skills folder:

```bash
cp -r .claude/skills/sdlc-studio ~/.claude/skills/
```

### How do I use the skill?

Run `/sdlc-studio help` for the full command reference, or `/sdlc-studio status` to see your current pipeline state.
