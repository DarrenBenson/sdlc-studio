- **The markdown-evidence guard derives what a verifier reads from the runner (BG0264).** Three versions of this guard re-parsed a verifier expression themselves, and each was defeated by an expression the runner reads differently: a directory glob, a flag counted as a target, a bare directory, and value-taking flags. It now asks `_build_command` for the argv that will actually run, drops operands the runner cannot read, and walks a directory because both `rg -q` and `grep -rqE` recurse. The earlier code asserted "grep without `-r` never reads a directory" - true of grep, false of this DSL, and the escape that assertion left open.
