# RETRO-0030: The bug-clearing sprint: four integrity fixes, and a migration gap surfaced

## Delivered

- **BG0142 (2pt) - reconcile agrees with check_links on what a valid link is.** Removed the dead
  type-dir fallback in `_link_exists`; an archive row now resolves only file-relative, so a
  regressed link can no longer hide in reconcile while check_links flags it (the LL0016 class).
- **BG0144 (3pt) - grooming refuses a fictional Affects.** A unit whose declared `Affects` paths
  ALL fail to resolve is ungroomed and the paths are named; a greenfield file the unit will create
  is tolerated as long as some path resolves. Gates both creation and planning from the one shared
  definition, so a typo is caught the moment the author can still fix it.
- **BG0145 (3pt) - churn risk survives for a docs unit.** `complexity.assess` folds an unscored
  file's churn-only risk into the risk band, so a constantly-churning doc is no longer invisible to
  the router; code `difficulty` stays `unknown`. The bug's part (1) - a `--seed-source` CLI
  restriction - was already removed by RFC0038, so it was declined, not fixed.
- **BG0146 (5pt) - a recalibration can no longer erase the evidence that justified it.**
  `sample_class` labels a row IN-SAMPLE only when the constants that MADE its forecast are the ones
  its actuals fitted. A row forecast by a retired estimator reads `stale-constants`, so RETRO0025's
  0.55x and RETRO0026's 0.39x stay honest falsifications, not "training error".

## Blocked / deferred

## What went well

- **All four fixes passed an independent adversarial review with no defects found.** The reviewer
  traced each fix's edge cases - the file-relative link resolution against both live and archive
  indexes, the churn-fold guard against double-counting, the all-vs-some Affects logic, and the
  sample_class reorder - and confirmed each behaves as specified.
- **Three of the four were the classes this project keeps re-learning** (two guards disagreeing,
  LL0016; a label contradicting its own number) and are now closed in code with regression tests.
- **The fixes replicated the discipline of the finale sprint:** real ACs with executable Verify
  lines, per-bug regression tests, and verification by an independent actor attacking the claim.

## What was hard / what stalled

- **BG0144 rippled into 12 test files.** Because the grooming gate runs at CREATION as well as
  planning (which the bug explicitly wanted - the filer refuses a fictional path at the moment the
  author can still fix it), every test that created a groomed unit with a placeholder `src/thing.py`
  Affects had to make that file real. ~140 test cases, swept by a subagent (which itself fanned out
  to parallel sub-agents), leaving the deliberate-refusal tests intact. This is the recurring
  lesson: a gate on the canonical creation path ripples into every fixture that used the old shortcut.
- **The sprint surfaced a release-blocking gap it could not fix in scope:** the whole two-backlog +
  sizing model is universal, not schema-gated, so an upgrading project would break. Captured as
  RFC0040 (see Lessons) rather than papered over.

## Lessons

- The migration story for a breaking model change is part of the change, not an afterthought. This
  session shipped a universal, non-schema-gated two-backlog + sizing model; an existing consuming
  project upgrading to it would find its CR workflow refused (G1), its childless-CR completion
  blocked (G2), and its stale-Affects units ungroomed - with no opt-in and no data migration.
  Because it is all unreleased (freeze until ~2026-07-21) there is a window to build the migration
  before it ships, and a breaking change earns a semver-major bump.
- A gate on the canonical creation path ripples into every test that used the old creation shortcut
  as a fixture (the Affects-resolve gate hit 12 files, ~140 cases). Budget the fixture migration as
  part of the gate.
- A bug's stated scope can be overtaken by an earlier change: BG0145 part (1) named a CLI
  restriction that RFC0038 had already removed. Verify the defect still reproduces before fixing the
  described symptom; decline with the reason when it does not.
