# EP0012: Distributed artefact identity (schema v3)

> **Status:** Draft
> **Created:** 2026-07-06
> **Created-by:** sdlc-studio new

## Summary

The schema-v3 foundation of v4: replace sequential ids with type-prefixed short ULIDs so
concurrent writers (trunk-based development, parallel agent worktrees) never collide, make
`_index.md` a fully regenerated output, and refresh the TRD that documents the changed write
contract. This is the migration every consuming project runs once, so it must be rock-solid
and reversible. Groups CR0167 (ULID ids + migration + aliases, from RFC0024), CR0168 (fully
derived indexes), CR0184 (TRD refresh). Foundational - the enforcement and workflow epics
build on the identity and index guarantees established here.
