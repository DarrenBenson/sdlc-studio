# Review-seat charter - superseded by the enriched seat schema

This template has been merged into the single seat schema. There is one seat schema now:
**`amigo-template.md`** (the enriched seat schema). It supersedes this charter and covers both
shapes of seat:

- a **build-capable** seat (a "Three Amigos" Engineering / QA / Product seat that both does the work
  and reviews it), and
- a **review-only** seat (a document owner such as Product Owner / Product Manager / UX that only
  critiques) - fill its review-render sections and mark the work-render sections "n/a".

Copy and customise `amigo-template.md` for any new review seat. This pointer remains so existing
references do not break.
