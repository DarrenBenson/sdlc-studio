**Lesson.** A subagent's report that a fix works will pass any reading, because the report was written by the thing that wants to be believed. The proof of the forecast-recording fix was not to read the code that reads the record: it was to DOUBLE both forecast constants in memory and check that no recorded estimate moved. That attack is the only thing that established the fix, and it took one command. Ask what the fix claims to make impossible, then try to do that thing. A guard that cannot be made to fail has not been shown to work - it has only been shown to be present.
